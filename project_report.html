Parent Eye - Project Analysis Report

Date: February 19, 2026
Project Type: AI-Powered Multilingual Parent Education Portal
Tech Stack: Python (Flask), JavaScript (Vanilla), Machine Learning (Scikit-learn)

1. Project Overview

Parent Eye is a web application designed to bridge the gap between schools and parents, particularly those who may not be fluent in English. It provides a real-time dashboard of student performance, attendance, and notifications, accessible in multiple Indian languages (English, Telugu, Hindi, Tamil).

The core innovation lies in its AI capabilities:

  • Performance Prediction: Uses machine learning to predict future marks based on historical trends.
  • Multilingual Chatbot: An intelligent assistant that answers parent queries in their native language, with voice support.

2. Technology Stack

Backend

  • Language: Python 3.x
  • Framework: Flask (Web Server)
  • AI/ML Libraries:
    • scikit-learn: Implementation of Linear Regression for marks prediction.
    • numpy: Numerical operations for data processing.
  • Natural Language Processing:
    • deep_translator (Google Translator): For translating dynamic content.
    • gTTS (Google Text-to-Speech): Server-side fallback for voice output.

Frontend

  • Core: HTML5, CSS3, Vanilla JavaScript (ES6+).
  • Visualization: Chart.js (Interactive line charts).
  • Voice API: Web Speech API for speech-to-text.

3. Key Features

A. Multilingual Support

Supports English, Telugu, Hindi, and Tamil. The app uses a hybrid approach with static dictionaries for UI elements and Google Translate API for dynamic content like messages and remarks.

B. AI-Powered Dashboard

The dashboard provides deep insights into student performance:

  • Visual Analytics: Color-coded marks and trend lines.
  • Predictive Engine: Uses Linear Regression (in models.py) to analyze historical data and forecast the next semester's performance, providing a confidence score and trend indicator (Improving/Declining).

C. Chat Assistant

The chatbot (in chatbot.py) is context-aware. It knows the logged-in student's data and can answer specific questions like "How is my child doing in Math?" in the parent's native language. It supports voice input/output for accessibility.
